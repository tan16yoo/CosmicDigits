# 🌌 CosmicDigits – Unlock Your Numerological Blueprint

**CosmicDigits** is a magical web app that calculates your **Life Path**, **Destiny**, and **Soul Urge Numbers** using your full name and birthdate — all based on ancient numerology principles. Instantly discover what your core numbers reveal about your personality and life path ✨

🔗 **Live Demo**: [https://tan16yoo.github.io/CosmicDigits/](https://tan16yoo.github.io/CosmicDigits/)

---

## 🔮 Features
- 🧾 Input your full name and date of birth
- 🔢 Calculate:
  - **Life Path Number**
  - **Destiny Number**
  - **Soul Urge Number**
- 📖 Get detailed meanings for all three
- 🎨 Aesthetic design with cosmic-inspired color palette
- ⚡ Fully responsive and mobile-friendly
